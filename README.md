# Opal: Shimmer

Shimmer is an application state and configuration management library built with Opal, a Ruby-to-JS compiler.

## Installation

Add this line to your application's Gemfile:

    gem 'opal-shimmer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install opal-shimmer
